<h2> 🐢 목이 거북해 </h2>

<br/>

### 🌿 프로젝트 소개
<p>
전자기기 사용량이 증가함에 따라, 거북목으로 고통받는 현대인들은 점차 늘어나고 있습니다. 🤕<br/>
거북목은 한번 진행되면 개선하기 어렵기 때문에, 사전에 방지하는 것이 매우 중요합니다. ⚠️<br/>
<b>목이 거북해</b>는 거북목 방지를 위해 <b>바른 자세 습관</b>을 길러주는 웹 서비스 입니다.<br/> 
목이 거북해와 함께 웹캠 만으로 손쉽게 바른 자세 습관을 길러보세요! 🐢
